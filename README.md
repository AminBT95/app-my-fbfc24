# FC24 Coach AI Flutter

Application Flutter mobile basée sur le plugin FC24DB Manager Pro.

Fonctions incluses :
- Banque joueurs depuis `assets/data/fc24-real-data.json`
- Comparateur joueur vs joueur
- Modes de duels avancés
- Poste vs poste
- Détection des joueurs plus forts que le joueur sélectionné
- CRUD joueur local simplifié
- Import nom depuis URL SoFIFA/FIFAIndex
- Terrain tactique animé en CustomPainter

## Lancer
```bash
flutter pub get
flutter run
```

## Build APK
```bash
flutter build apk --release
```
