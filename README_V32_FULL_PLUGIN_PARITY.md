# FC24 Coach AI Flutter Pro — v32 FULL PLUGIN PARITY

## Ajouts v32

### Team vs Team
- Nouvelle section **À éviter joueur par joueur**.
- Analyse basée sur les duels proches entre joueurs du XI.
- Conseils automatiques : éviter duel physique, course longue, 1v1 sans couverture, conduite trop longue, centres aériens, etc.
- Score par duel et mode tactique associé.

### Managers Coach
- Page dédiée aux managers.
- Style probable : attaque/pressing, bloc solide, équilibré.
- Système conseillé : 4-3-3, 4-2-3-1, 5-3-2 selon profil équipe.
- Hidden context : profondeur, rythme, impact physique.

### Formation Counter Engine
- Contres de formations : 5-3-2, 4-3-3, 4-2-3-1, 4-4-2, 3-5-2.
- Plan attaque/défense par système.
- Ce qu’il faut éviter selon la formation adverse.

### Situations Coach
- Espace LB-CB/RB-CB.
- Cutback.
- Bloc bas.
- Pressing haut.
- Milieux qui montent trop.
- Erreurs fréquentes FC24.

### Stats Encyclopedia
- Explication des stats importantes selon les duels.
- Acceleration, Sprint Speed, Agility, Balance, Strength, Def Awareness, Interceptions, Standing Tackle, Composure, Reactions, Positioning, Heading/Jumping.

### Duel Visuals CRUD
- Création/suppression de templates de duel.
- Mode duel lié à chaque scénario.
- Notes/animation pour préparer les futurs scénarios visuels.
