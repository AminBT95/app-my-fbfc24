# V19 Comparator All Modes

Ajouts :
- Comparator Pro+ conserve le design bleu/navy/teal apprécié.
- Tous les modes de comparaison sont affichés sur une seule page.
- Chaque mode affiche score A/B, gagnant A/B/égalité et barre de résultat.
- Cliquer sur un mode ouvre un détail complet : stats utilisées, poids, valeurs joueur A/B, score core/profil/playstyles, lecture coach, comment profiter et contrer.
- Bilan global en bas : total de victoires joueur A, joueur B et égalités sur tous les modes.
- Contraste renforcé : zone résultats en fond dark navy avec textes blancs/bleu clair/vert lisibles.

Build :
flutter clean
flutter pub get
flutter build apk --release
