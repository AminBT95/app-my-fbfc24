# V33 — IA Simulator parity + Team Analyzer Pro+

Demandes traitées :

- Team vs Team enrichi depuis la logique IA Simulator du plugin WordPress :
  - scénario home/away : opponent weaker / equal / stronger,
  - phases : build-up, pressing adverse, transition, bloc bas,
  - instructions terrain automatiques,
  - comparaison par couloir gauche/droite, axe, surface, transition défensive,
  - conseils de ce qu’il faut attaquer et éviter selon les zones.

- Détection meilleurs joueurs :
  - les joueurs trouvés sont cliquables,
  - ouverture d’un modal de comparaison complet contre le joueur référence,
  - score, critères pondérés, PlayStyles et lecture coach visibles.

- Matchup Finder :
  - clic joueur ouvre un modal de comparaison contre un adversaire naturel proche.

- CRUD Players :
  - ajout stats globales : PAC, SHO, PAS, DRI, DEF, PHY,
  - ajout Att. Positioning, Marking, Block,
  - ajout Volleys, Penalties,
  - ajout stats GK : Diving, Handling, Kicking, Positioning, Reflexes,
  - stats organisées par groupes repliables.

- Team Analyzer :
  - rapport par phase : vitesse/profondeur, build-up, bloc défensif, aérien, pressing,
  - weak links fonctionnels,
  - instructions tactiques recommandées selon profils joueurs,
  - joueurs clés restent cliquables.
