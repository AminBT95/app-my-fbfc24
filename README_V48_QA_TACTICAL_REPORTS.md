# FC24 Coach AI Pro v48 — QA + Tactical Lab + Reports

## Objectif
Version de finalisation après la v47 : stabiliser l'app, compléter Tactical Lab, ajouter Reports/QA et rendre les exports plus faciles.

## Ajouts principaux
- Tactical Lab Pro V48 : terrain drag & drop, sélection joueurs, timeline, simulation duel, bibliothèque tactique, export rapport texte/JSON.
- Reports / QA : nouvelle rubrique dans le menu avec rapports, backup DB complet, checklist stabilité et performance.
- VIP Final Center V48 : dashboard premium, favoris persistants, workflow coach, heatmaps, rapports.
- Backup DB : export JSON complet joueurs, équipes, idées tactiques, historique et favoris.
- QA intégrée : vérification Player Detail, PlayStyles/traits, favoris, Team vs Team, exports.

## Notes
- Les exports PDF/GIF/vidéo natifs restent simulés en texte/JSON pour éviter d'ajouter des dépendances lourdes et risquer de casser le build.
- Les rapports sont copiables via le presse-papiers.
