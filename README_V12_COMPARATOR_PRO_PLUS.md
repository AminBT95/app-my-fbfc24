# FC24 Coach AI Flutter — V12 Comparator Pro Plus

Cette version renforce le comparateur pour se rapprocher de la version HTML/CSS/JS `Duel Engine Pro Plus`.

## Ajouts principaux
- Comparateur Duel Engine Pro+ plus riche
- Modes avancés : pressing haut, passe vs interception, sortie de balle, cutback, contre-attaque, pivot, contrôle orienté, jockey, deuxième ballon, ailier inversé, frappe de loin
- Matchups poste vs poste : ST vs CB, ST vs GK, RW vs LB, LW vs RB, ailier vs CB, CAM vs CDM, CM vs CM, centre/tête
- Comparaison détaillée : taille, poids, body type, AcceleRATE
- Différences de stats avec barres
- Diagnostic tactique automatique
- Impact PlayStyles + profil physique
- Détail du calcul pondéré avec poids par stat
- Ajout modes : dribble couloir, dribble axe, GK 1v1, frappe de loin

## Note data
Les résultats dépendent toujours de la qualité du fichier `assets/data/fc24-real-data.json`. Si certains traits/playstyles sont absents du JSON source, l’app ne peut pas les inventer.
