# FC24 Coach AI Pro v11 — UI/Data/Filters/Teams

Ajouts principaux :
- Nouvelle base joueurs avec recherche avancée.
- Female teams et Soccer Aid cachés par défaut, réactivables via filtres.
- Filtres riches : poste, équipe, pied, AcceleRATE, OVR, vrais noms, PlayStyles/Traits, GK only.
- Fiche joueur modernisée avec header premium, miniface, badges OVR/POT/SM/WF.
- Stats catégorisées : principales, GK, vitesse/physique, tir, passe/création, dribble, défense.
- Stats GK ajoutées dans l’affichage détaillé.
- Teams cliquables : fiche équipe, formation terrain, analyse tactique, effectif cliquable.
- Team XI corrigé : lecture des IDs depuis la liste `xi` de la database.

Note : les noms et certains PlayStyles dépendent encore de la qualité de `fc24-real-data.json`. Cette v11 améliore l’UI, les filtres et l’exploitation des champs présents.
